# TrainLight-Lego

An arduino project for kids

Can detect lego train is coming by ultrasonic (codes/ -ultra.ino) or controlled by rip/button (codes\-rip.ino).

**Reference:**

Ultrasonic sensor:

https://dronebotworkshop.com/hc-sr04-ultrasonic-distance-sensor-arduino/
